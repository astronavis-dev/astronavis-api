# Astronavis API

REST API server for the Astronavis galactic cartography and navigation platform.

## Planned features

- planetary search
- galactic routing
- nearby systems lookup
- route explanation
- synchronization services
- OpenAPI/Swagger support

## Tech stack

- Rust
- Axum
- Tokio
- SQLx
- SQLite/PostgreSQL
